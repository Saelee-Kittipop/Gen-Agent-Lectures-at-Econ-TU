# Notebooks

## วิธีใช้

### Google Colab (แนะนำ)
1. Upload notebook ไปที่ Google Drive
2. Double-click เปิดด้วย Google Colab
3. ตั้ง `USE_REAL_LLM = False` (default)
4. Run All!

### Jupyter Notebook ในเครื่อง
```bash
pip install jupyter
jupyter notebook
```
